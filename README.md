# Anidaku

A professional anime streaming application built with Kotlin and Jetpack Compose, powered by the Miruro API.

## Features
- Professional and clean UI
- Trending anime discovery
- Advanced search functionality
- Detailed anime information
- Episode listings

## Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Networking**: Ktor
- **Image Loading**: Coil
- **Serialization**: Kotlinx Serialization
- **API**: [Miruro API](https://miruro-api-rho.vercel.app/)

## Colors
- Primary Background: #000000 (Black)
- Primary Text: #FFFFFF (White)
- Secondary Text: #808080 (Gray)

## Build
The project uses Gradle and follows the standard Android project structure.
